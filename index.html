Requirements

  • Python 2.4, 2.5, 2.6, 2.7, 3.2 or 3.3
  • Linux, OSX, Cygwin or other unix-like OS
  • (optional) ncurses for curses_display
  • (optional) python-glib for GlibEventLoop
  • (optional) Twisted for TwistedEventLoop
  • (optional) Apache for web_display module
